# Loch Project

A modern React + Chakra UI web app for Loch

## Features

- Notification carousel with infinite looping
- Responsive
- Signup form
- Beautiful, modern UI

## Getting Started

### Prerequisites

- Node.js (v16 or newer recommended)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone <your-repo-url>
   cd <your-repo-folder>
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the App (Development)

```bash
npm run dev
# or
yarn dev
```

The app will be available at [http://localhost:5173](http://localhost:5173) by default (Vite).

### Building for Production

```bash
npm run build
# or
yarn build
```

### Previewing Production Build

```bash
npm run preview
# or
yarn preview
```

### Type Checking

```bash
npm run type-check
# or
yarn type-check
```

### Linting

```bash
npm run lint
# or
yarn lint
```

## Project Structure

- `src/components/` — All React components (carousel, testimonials, signup form, etc)
- `src/App.tsx` — Main app layout
- `public/` — Static assets (images, icons, etc)

## Technologies Used

- React
- Chakra UI
- Framer Motion
- react-multi-carousel
- TypeScript
- Vite
